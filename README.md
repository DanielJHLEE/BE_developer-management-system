# developer-management-system
개발자 인력 관리 프로젝트

Spring Boot, JPA, Gradle, Swagger UI를 사용하여 개발자 인력 정보를 관리하는 API

# 개발 환경 Backend
* Java (JDK 17)
* Spring Boot 3.x
* Spring Data JPA
* Lombok
* Gradle
* Swagger (Springdoc OpenAPI)

# GRADLE 명령어 빌드/실행
* ./gradlew clean build 빌드  
* ./gradlew bootRun
